# 🌧️ South India Rainfall Tracker (2021–2025)

## 🚀 Project Overview

The **South India Rainfall Tracker** is an interactive data analytics project built using Python to explore rainfall patterns across South Indian states from **2021 to 2025**. The project focuses on applying Exploratory Data Analysis (EDA) techniques and geospatial visualization to transform raw data into meaningful insights.

Rather than limiting the analysis to basic summaries, this project emphasizes understanding trends, comparing regional variations, and presenting findings through interactive visualizations that make the data easier to interpret.

## ✨ Key Features

- 📊 Interactive visualization of rainfall data across multiple years
- 🗓️ Year-wise comparison using a time-slider interface
- 🗺️ Geospatial mapping for intuitive state-level analysis
- 🧹 Data cleaning and preprocessing using Pandas
- 📈 Clear visual representation of rainfall trends and patterns

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Folium
- Matplotlib
- Jupyter Notebook

## 💡 Skills Demonstrated

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Geospatial Data Visualization
- Trend Analysis and Pattern Recognition
- Interactive Dashboard Development
- Data Storytelling and Insight Generation

## 🎯 Key Learnings

This project highlights the importance of going beyond simple descriptive statistics to explore data in a structured and meaningful way. AI-assisted tools were used to streamline repetitive development tasks, allowing greater focus on analysis, interpretation, and visualization while maintaining full responsibility for validating results and drawing conclusions.

## 📌 Outcome

The final output is an interactive rainfall tracker that enables users to explore historical rainfall patterns across South India in an engaging and informative way. The project demonstrates practical skills in Python-based data analysis, visualization, and communicating insights from real-world datasets.
